graphql-java Subscriptions

graphql-java subscriptions使用websocket不断更新商品价格

Explanation

这里模拟是的一个库存变化的，一共有4种:

  1. TEAM
  2. IBM
  3. AMZN
  4. MSFT
  5. GOOGL
订阅其中的TEAM，IBM两种,其中价格上升时显示为绿色，下降显示红色，注意查看

当浏览器客户端订阅，服务端使用rxjava2/spring reactor实现的推送系统服不断的推送数据更新

使用的查询例子如下 :

请求
subscription stockFluxQuotes($stockCodes: [String]) {
  stockFluxQuotes(stockCodes: $stockCodes) {
    dateTime
    stockCode
    stockPrice
    stockPriceChange
  }
}
参数
{
  "stockCodes": ["TEAM", "IBM"]
}

库存价格更新
